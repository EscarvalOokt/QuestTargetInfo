# Changelog

## v2.0.0

### Added

    * Added travel information for selected world objects.
    * Added travel information for selected empty world tiles.
    * Added a `Travel` tab to the world inspect pane.
    * Added route information with distance, current-map handling and planet layer context.
    * Added range display for supported transport types.
    * Added clearer fuel information for transport pods, shuttles and gravships.
    * Added shuttle outbound, return and total fuel information.
    * Added estimated shuttle launch count.
    * Added support for showing unavailable or out-of-range transport sections.
    * Added status messages for unavailable routes, missing vehicles, invalid targets, invalid landing sites, missing fuel and signal jammer requirements.
    * Added compact mode for a denser travel information display.
    * Added settings to show or hide the quest travel window and world inspect `Travel` tab.
    * Added settings to show or hide individual transport sections.
    * Added settings for unavailable transports, ancient pod compatibility and layer-adjusted distance display.

### Changed

    * Expanded the mod from quest-only travel info to broader world-map travel info.
    * Improved the quest travel window with a clearer route and transport breakdown.
    * Improved transport sections for transport pods, shuttles and gravships.
    * Improved range and fuel labels to make travel requirements easier to read.
    * Improved handling of Odyssey-specific travel information when the DLC is active.
    * Improved support for targets across planet layers.
    * Improved behavior for targets that cannot be reached, landed on or used as transport destinations.

### Localization

    * Expanded localization coverage for new travel information, transport statuses and settings.
    * Updated English, German, Russian and Ukrainian translation files.
